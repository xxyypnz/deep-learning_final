      46              :  * We can transform this into

      48              :  *                      AND (a.x = 42 OR a.x = 44)

      51              :  * perhaps as index qualifications, and in any case reducing the number of
      52              :  * rows arriving at the join.  In essence this is a partial transformation to

      54              :  * unravel the original OR --- doing so would usually bloat the qualification
      55              :  * expression to little gain.
